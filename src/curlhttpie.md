# curl/httpie

## curl

- Upload form file: 
  ```
  curl -F ‘data=@path/to/local/file’ UPLOAD_ADDRESS
  ```
- Add user agent
  ```
  curl -A "user-agent-name-here" [URL]
  e.g. "Mozilla/5.0 (X11; Linux x86_64; rv:60.0) Gecko/20100101 Firefox/81.0"
  ```
